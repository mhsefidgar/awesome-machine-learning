## Datasets

A curated list of open-source datasets for machine learning projects.

---

### General Machine Learning
- [**UCI Machine Learning Repository**](https://archive.ics.uci.edu/ml/index.php) – Classic datasets for regression, classification, and clustering.
- [**Kaggle Datasets**](https://www.kaggle.com/datasets) – Diverse datasets spanning all ML tasks.
- [**OpenML**](https://www.openml.org/) – Large collection of datasets, usable with Python and R.
- [**Google Dataset Search**](https://datasetsearch.research.google.com/) – Aggregates freely available datasets from across the web.

---

### Computer Vision
- [**MNIST**](http://yann.lecun.com/exdb/mnist/) – Handwritten digit classification.
- [**Fashion-MNIST**](https://github.com/zalandoresearch/fashion-mnist) – Clothing image classification.
- [**CIFAR-10 & CIFAR-100**](https://www.cs.toronto.edu/~kriz/cifar.html) – Object recognition in 10 or 100 classes.
- [**COCO (Common Objects in Context)**](https://cocodataset.org/) – Large-scale object detection, segmentation, and captioning.
- [**ImageNet (research subset)**](http://www.image-net.org/) – Visual object recognition dataset.

---

### Natural Language Processing (NLP)
- [**IMDB Movie Reviews**](https://ai.stanford.edu/~amaas/data/sentiment/) – Sentiment analysis dataset.
- [**20 Newsgroups**](http://qwone.com/~jason/20Newsgroups/) – Text classification dataset.
- [**WikiText**](https://blog.einstein.ai/the-wikitext-long-term-dependency-language-modeling-dataset/) – Language modeling dataset.
- [**Common Crawl**](https://commoncrawl.org/) – Large-scale web crawl dataset for NLP tasks.
- [**SQuAD (Stanford Question Answering Dataset)**](https://rajpurkar.github.io/SQuAD-explorer/) – Question answering dataset.

---

### Time Series / Forecasting
- [**M4 Competition Dataset**](https://www.m4.unic.ac.cy/) – Yearly, quarterly, and monthly time series.
- [**Yahoo Finance Stock Data**](https://finance.yahoo.com/) – Historical financial market data.
- [**UCI Time Series Datasets**](https://archive.ics.uci.edu/ml/datasets.php?format=&task=ts&att=&area=&numIns=&type=&sort=nameUp&view=list) – Various time series datasets.

---

### Audio / Speech
- [**LibriSpeech**](http://www.openslr.org/12/) – Speech recognition dataset from audiobooks.
- [**VoxCeleb**](http://www.robots.ox.ac.uk/~vgg/data/voxceleb/) – Speaker identification dataset.
- [**FreeSound**](https://freesound.org/) – Open audio samples for sound classification.

---

### Reinforcement Learning
- [**OpenAI Gym Environments**](https://gym.openai.com/) – Simulated environments for RL experiments.
- [**DeepMind Control Suite**](https://www.deepmind.com/research/open-source/deepmind-control-suite) – Continuous control RL tasks.

---

### Healthcare / Biology
- [**PhysioNet**](https://physionet.org/) – Biomedical signals and clinical data.
- [**Human Protein Atlas**](https://www.proteinatlas.org/) – Protein expression images.
- [**TCGA (The Cancer Genome Atlas)**](https://www.cancer.gov/about-nci/organization/ccg/research/structural-genomics/tcga) – Genomics datasets (open access portions).

---

### Tabular / Structured Data
- [**Titanic Survival Dataset**](https://www.kaggle.com/c/titanic/data) – Classic dataset for binary classification tutorials.
- [**Adult Income Dataset**](https://archive.ics.uci.edu/ml/datasets/adult) – Predict if income exceeds \$50k/year.
- [**Wine Quality Dataset**](https://archive.ics.uci.edu/ml/datasets/Wine+Quality) – Regression and classification dataset.
